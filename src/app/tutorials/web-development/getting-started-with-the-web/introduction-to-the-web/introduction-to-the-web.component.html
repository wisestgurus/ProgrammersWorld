the web

The web, by definition is just a combination of different websites linked together and shared over the internet.

Note: A website links two or more webpages together, but the web links two or more websites together. Please try and understand the difference.
brief analogy of the web

In a library consisting of millions of books, each book has its own number of pages. Depending on the book, a page might have images in it, some might include icons in it as well, and in general the text of every page in a book is styled nicely.

Now read carefully;
  • The library consisting of millions of books is equivalent to the web.
  • Each page of a book is equivalent to a webpage.
  • Each book is equivalent to a website.
  • The images, icons or font types used in a particular page of a book is equivalent to a web resource (or an asset).
  • To be able to search and find books in the library (or search and find websites in the web), you will need a search engine.
  • To be able to view the book you searched for (or view the website you searched for), you will need a web browser.

the internet

The internet by definition, is just a network of networks. This is what makes connection and sharing of data on the web globally possible.

brief analogy of the internet
Continuing from the library example above. Remember the library consist of different books (which is equivalent to the web consisting of different websites), but how can people get good delivery after buying those books. Fortunately, the library has a delivery person which delivers the books to those who buy it.
Now read carefully;
  • The delivery person is equivalent to the internet, because he/she connects two bodies or networks together (e.g. the library with upto 1 million buyers together) and deliver the books successfully.

the web vs the internet

Comparison between the web and the internet
The web The internet
The web is a combination of different HTML documents with all its web resources linked together. The Internet is a network which connects other networks together to allow for transmission of data within themselves possible. This is what makes documents on the web visible globally.
The web is similar to a library of different books. The Internet is similar to a delivery person.

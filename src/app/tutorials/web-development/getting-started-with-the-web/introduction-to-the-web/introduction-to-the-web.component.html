the web

An information system made up of different websites linked together is referred to as the web. (An information system in a layman term is anything designed to collect, process, store, and distribute information).

A website links two or more webpages together, but the web links two or more websites together. Please try and understand the difference.
brief analogy of the web

In a library consisting of millions of books, each book has its own number of pages and every book is recognized by its unique name. Depending on the book and the author, a book can include images in some its pages, some only have text in it, some can be well descriptive with lots of illustrations, some might include icons in it and in general the text of every page in a book is styled nicely based on the taste of the author.

Now read carefully;
  • The library (an information system) consisting of millions of books is equivalent to the web consisting of different websites.
  • Each book is equivalent to a website with a unique name.
  • Each page of a book is equivalent to a webpage of a website.
  • The images, icons or font types used in a particular page of a book is equivalent to a web asset included in a webpage.
  • To be able to search and find books in the library (or search and find websites in the web), you will need a search engine.
  • Inside of an application consists of a search engine (with the search bar being the visible part) and anything related to the web. This is a web browser.

the internet

The internet by definition, is a global system of interconnected networks. This is what makes connection and sharing of contents on the web globally possible. (A network is when two or more things connect together).

The moment you on your mobile data, you are automatically connected to the internet. Other devices which provide internet access include; a router, a modem, a wifi, etc...

brief analogy of the internet

Continuing from the library example above. People generally visit the library to buy books and some prefer to order. Fortunately enough, the library has a delivery man responsible for delivering various books successfully, to whomsover orders.

Now read carefully;
  • The delivery man connects the library with billions of buyers (i.e. he serves as an interemediary).
  • The internet is similar to a delivery man. The only difference being that the internet is an intermediary network with much broader connectivity (i.e. can connect someone from Nigeria, to someone in UK, with another person in US, with another person in Nigeria again (and so on and so forth) bonding them all together in its own network for sharing of data or resources).
  • The delivery person is equivalent to the internet, because he/she connects two or more bodies or networks together (e.g. the library with upto 1 million buyers together) and delivers the books successfully.

the web vs the internet

Comparison between the web and the internet
The web The internet
The web is a combination of websites linked together and shared over the internet. The Internet is an intermediary network of interconnected networks.
The web is just an information system consisting of series of websites made up of HTML documents with all its resources (e.g. web assets, etc). Nothing more. The internet is what makes connection and sharing of contents on the web globally possible.
The web is similar to a library of different books. The Internet is similar to a delivery person.

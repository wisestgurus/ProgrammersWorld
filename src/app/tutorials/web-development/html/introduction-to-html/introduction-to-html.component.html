html

HTML stands for HyperText Markup Language.

It is the document used by the web to display static contents.

It deals with the structure and display of webpages.

It consists of tags, elements and attributes as the basic building blocks.

With HTML alone, you only get static contents displayed (i.e. content that doesn't change), with no visually appealing look.

It is usually assisted with other technologies like, CSS (Cascading StyleSheets) - for styling the looks and feel of a webpage and JavaScript (Scripting Language) - for dynamically updating the content of a webpage (opposite of static contents - i.e. contents that change based on an event).

You will learn more about CSS and JavaScript later on in their respective tutorial section.

html tags

A HTML tag or a tag is a markup syntax surrounded by angle brackets (<>).

Tags are not displayed by web browsers, but used to render the content of a webpage.

HTML tags come in pairs;

  1. An opening tag (<tagname>) also called a start tag and
  2. A closing tag (</tagname>) also called an end tag.

With HTML tags alone, nothing gets displayed on the web browser. This is were HTML Elements come to play.

html elements

HTML elements or elements are blocks of code which consist of either;

  1. An opening tag (<tagname>), element content and a closing tag(</tagname>), or
  2. An opening tag (<tagname>), with no element content. This is called an empty element; written as (<tagname>) or (<tagname/>)

Empty elements are also called empty tags or void elements. It doesn't contain any element content and are usually closed in the start tag like this;(<tagname/>).

Empty tags or elements must not be closed, but it's recommended for strict validation process. At ProgrammersWorld, we highly recommend closing of empty tags.

HTML elements can be nested (i.e. elements can contain elements).

html attributes

A HTML attribute or an attribute provides additional information about an element.

All HTML elements can have attributes.

HTML attributes usually come in a name/value pairs, like; name='value'.

HTML attributes are always specified in the start tag.

Since HTML tags are not displayed by web browsers and an attribute is specified inside of a tag, we can say that attributes too, are not displayed by web browsers.
